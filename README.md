## 클래식 공연 예매 시스템

### 기능
1. 공연 예매
2. 공연 생성, 조회, 수정, 삭제
3. 공연 리뷰 생성, 조회, 수정, 삭제
4. 좌석평 생성, 조회, 수정, 삭제
5. 회원가입, 로그인, 로그아웃

### 개발 도구
GUI: MFC(Microsoft Foundation Classes) 

DB: SSMS(SQL Server Management Studio)
